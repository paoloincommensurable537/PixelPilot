## 🛠️ Troubleshooting Common Issues

Most users find the setup process simple, but some minor issues may occur.

**The AI ignores the rules.**
Check that you saved the rule text in the correct configuration file for your specific editor. Restart the editor to ensure it loads the new settings. Verify that you pasted the full list of rules without any gaps.

**The code still looks messy.**
Ensure you are using the latest version of your AI coding tool. Older versions may not support advanced prompt instructions. Sometimes, a very broad request causes the AI to lose track of the rules. Try to write specific instructions, such as, "Build a contact form using the PixelPilot standards."

**The installer does not start.**
Ensure your Windows system is up to date. Occasionally, system security settings block unknown files. Right-click the installer and choose Run as Administrator if your computer stops the process.
