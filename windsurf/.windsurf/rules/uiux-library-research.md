# UI/UX Library Research 2026

> Research process for defining requirements, searching for options, 
> evaluating candidates, and providing recommendations.

---

## 1. WHEN TO USE THIS SKILL

- The user asks for a library recommendation.
- Existing skills reference a potentially outdated library.
- A new project requirement emerges that isn't covered by existing skills.

---

## 2. RESEARCH PROCESS

### Step 1: Define Requirements
- What is the core problem?
- What is the tech stack (React, Vue, Vanilla JS)?
- What is the design language?
- What are the performance and accessibility needs?

### Step 2: Search for 2026 Options
- Use `search` tool with queries like: "best [category] library 2026", "modern [category] react library", "[category] alternatives to [old-library]".
- Look for GitHub repositories, npm packages, and community discussions.

### Step 3: Evaluate Candidates
- **GitHub Stars**: Community popularity.
- **npm Downloads**: Real-world usage.
- **Bundle Size**: Impact on performance.
- **Framework Support**: Compatibility with the project's stack.
- **Performance**: Benchmark data (if available).
- **License**: MIT, Apache 2.0, etc.
- **Documentation**: Quality and clarity of examples.

---

## 3. PROVIDE A RECOMMENDATION

Top 2-3 options, each with a "best for" use case.

| Library | Best For | Pros | Cons |
|---------|----------|------|------|
| **Library A** | Large apps | Feature-rich, well-documented | Heavy bundle size |
| **Library B** | Small apps | Lightweight, fast | Limited features |
| **Library C** | Experimental | Cutting-edge features | Potential instability |

---

## 4. UPDATING SKILL FILES

If a library becomes a new industry standard:
- Propose updating the relevant skill file (e.g., `uiux-admin-dashboard.md`).
- Include the new library in the "Decision Guide" or "Recommended Libraries" section.
- Provide a code example for the new library.

---

## 5. EXAMPLE RESEARCH TASK

**Task**: Find a modern 2026 library for data visualization in React.

1. **Requirement**: React, performance-critical, modern look.
2. **Search**: "best react chart library 2026".
3. **Candidates**: Recharts, Tremor, AG Charts.
4. **Evaluation**: Recharts (High popularity, React-native feel), Tremor (Tailwind-first, fast setup), AG Charts (Enterprise performance).
5. **Recommendation**: Recharts for most cases, Tremor for Tailwind projects.
